## 项目描述
简要描述这个项目的目的和功能。
## 环境依赖
- Python 2.7
- Jdk 1.8
## 安装流程
1. 克隆本仓库
git clone https://github.com/kevinJhz/eventchains-aaai.git
2. 安装依赖
cd ./main/lib
pip install -r requirements.txt
3. 执行以下命令进行训练/测试/使用
bash 
# 训练模型
python train.py

# 测试模型 
python test.py

# 使用模型进行预测 
python use.py
